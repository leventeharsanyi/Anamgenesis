# Anamgenesis

Small Streamlit demo app using **Python 3.12** and **Poetry**.

## Requirements
- Python 3.12
- Poetry
- Git

## Setup

```bash
git clone <your-repo-url>
cd anamgenesis
poetry env use python3.13
poetry shell
poetry install
```

## Running the app

```bash
poetry run streamlit run app/main.py
```
